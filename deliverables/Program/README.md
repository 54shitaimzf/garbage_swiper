# Program 程序交付说明

源码保持原目录，不整体移动到 `Program/`，以免破坏现有启动脚本、相对路径和车端部署路径。

| 目录 | 交付内容 |
| --- | --- |
| `web/` | 8081 手动控制前端 |
| `jetson/` | 8081/8082 车端网关和启动脚本 |
| `autonomy/` | 8082 PWA 与任务 API |
| `edge/` | TensorRT 推理入口 |
| `ros2_ws/` | ROS2 感知和传感器健康检查源码 |
| `cloud/` | 可选 MQTT 接口 |
| `scripts/` | 训练、推理和验证脚本 |
| `tests/` | 自动化测试 |
| `android/` | 可选客户端历史源码；当前主入口为 PWA |
