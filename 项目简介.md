# 项目简介 — 园区智能无人巡检环卫终端

> 面向**评委 / 非技术读者**的一页纸项目介绍。技术细节见 `实用文档/`，演示脚本见 `05-演示方案/`。

---

## 一、我们做了什么

我们构建了一个**园区智能无人巡检环卫终端**：一辆搭载 Jetson 边缘算力的无人小车，能够在园区（校园 / 产业园）道路自主巡逻，自动识别路面垃圾、异物、积水等**环卫事件**，发现异常即时本地报警、上报云端，并通过**鸿蒙（HarmonyOS）终端**推送给管理员，任务结束后自动返航。

系统以 **ROS2（Foxy）** 为机器人系统底座，以**鸿蒙 ArkTS 应用**为人机交互终端，形成"**终端下发任务 → 自主导航巡检 → AI 识别 → 异常上云告警 → 自动返航**"的完整闭环。

## 二、为什么做这个

- **场景真实有价值**：园区道路环卫巡检目前大量依赖人工，存在盲区与时效问题；无人巡检可 7×24 覆盖、降本增效。
- **技术栈前沿且自主可控**：ROS2 机器人框架 + 国产鸿蒙操作系统终端，契合"自主可控 / 信创"方向。
- **闭环完整、演示友好**：从任务下达到返航一气呵成，适合课程答辩与路演展示。

## 三、核心能力（亮点）

| 能力 | 说明 |
|---|---|
| 自主巡检 | 基于 Nav2 沿预设园区路线自动巡航、避障、逐点巡检、自动返航 |
| AI 道面识别 | YOLO 模型识别垃圾 / 异物 / 积水，边缘 TensorRT 加速（Orin Nano 20 TOPS） |
| 鸿蒙人机终端 | 鸿蒙 App 一键下发任务、实时查看状态、接收告警推送、手动接管 |
| 云端协同 | 异常事件经 MQTT 上报云端控制台，形成可追溯的环卫事件记录 |
| 多重安全 | 显式状态机 + 看门狗 + 急停 + 低压告警，运行可靠 |

## 四、技术方案（一句话版）

- **硬件**：Jetson Orin Nano + 思岚 A1 激光雷达 + 奥比中光 Astra 深度相机 + 8 路 IoT 传感器 + AT32 底盘。
- **系统**：ROS2 Foxy（机器人系统）+ Nav2（导航）+ YOLO/TensorRT（视觉）+ rosbridge（终端通信）+ MQTT（上云）。
- **终端**：鸿蒙 HarmonyOS ArkTS 应用，经 WebSocket 与车端通信。
- **部署**：Docker 容器化 + GitHub Actions CI，一键上车运行。

## 五、项目定位与边界（诚实说明）

本项目是**课程实训阶段的最小可行产品（MVP）/ 原型系统**，目标是验证完整技术链路与可演示闭环。当前阶段：仿真与容器内联调已通，真实 YOLO 推理模型、自建巡检地图、鸿蒙终端应用为明确标注的后续集成项；真车现场测试为后续执行项。详细模块完成度见 `../00-项目叙事与定位.md` 诚实状态表。
